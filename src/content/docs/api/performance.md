:::note[Проверено в игре]
Раздел проверен 15 июля 2026 года в KILLSCRIPT Pre-Alpha. API доступен только в клиентской Lua-части.
:::

`Performance` предоставляет усреднённые показатели FPS, времени кадра и загрузки CPU/GPU. В `server.lua` Reflex-модуля глобальный объект `Performance` равен `nil`.

## Быстрый пример

```lua
print("FPS: " .. Performance.AvgFps)
print("Max frame: " .. Performance.MaxFrameMs .. " ms")
```

## Свойства

| Свойство | Тип | Доступ | Описание |
|---|---|---|---|
| `AvgFps` | `number` | `get` | Средний FPS за последнюю секунду. |
| `CpuUsagePercent` | `number` | `get` | Средняя загрузка CPU игрой за последнюю секунду относительно времени кадра. |
| `GpuUsagePercent` | `number` | `get` | Средняя загрузка GPU игрой за последнюю секунду относительно времени кадра. |
| `MaxFrameMs` | `number` | `get` | Максимальное время кадра за последнюю секунду, в миллисекундах. |

:::caution[Все свойства getter-only]
Старая документация ошибочно помечала четыре свойства как `get/set`. В игре запись в каждое из них отклоняется Lua access error.
:::

## Интерпретация значений

Показатели динамические и описывают недавний интервал, а не весь матч. Во время одной проверки были получены `AvgFps ≈ 144.83`, `CpuUsagePercent ≈ 99.96`, `GpuUsagePercent ≈ 87.81` и `MaxFrameMs ≈ 20.76`.

Не используйте эти числа как ожидаемые значения — они зависят от устройства, настроек и текущей нагрузки.

## Пример предупреждения о низком FPS

```lua
Scheduler:OnFrame(function()
    if Performance.AvgFps < 30 then
        -- Показать предупреждение в интерфейсе
    end
end)
```
