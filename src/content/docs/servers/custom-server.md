Пользовательский сервер удобен для разработки: можно выбирать карту, перезапускать матч и проверять [Reflex](../../reflex/architecture/) без матчмейкинга.

## Создание

1. Откройте раздел пользовательского сервера в главном меню.
2. При необходимости укажите ссылку на собственную игровую конфигурацию.
3. Создайте сервер и дождитесь подключения.
4. Откройте меню паузы, чтобы скопировать код приглашения.
5. Передайте код участникам, которым доверяете.

Lua-модули начинают выполняться после загрузки игровой сессии.

## Права

Команды сервера имеют уровни доступа:

- **Anybody** — общие команды;
- **Host** — владелец пользовательского сервера;
- **Admin** — внутренние административные и диагностические команды.

Консоль показывает только команды, разрешённые текущему игроку и режиму матча. Поэтому набор доступных команд может отличаться от пользовательского сервера в matchmaking.

## Использование для модулей

Рекомендуемый цикл:

1. включите нужный обычный модуль;
2. выберите один Reflex-модуль, если он нужен;
3. создайте сервер;
4. откройте консоль `~` и выберите фильтр модуля;
5. сохраните исходники и дождитесь hot reload;
6. используйте команды управления раундом для повторяемого сценария.

Для проверки client/server обмена смотрите сообщения обеих частей в логе модуля.

## Пользовательская конфигурация

При создании сервера интерфейс принимает ссылку на конфигурацию игры. Такая конфигурация может менять параметры предметов, глобальные константы и флаги механик.

После изменения подключённой конфигурации команда:

```text
reload
```

перезагружает игровую базу сервера. Изменение конкретного параметра может потребовать перезапуска карты или раунда.

Актуальные имена параметров доступны через [GameConfig](../../api/game-config/). Не используйте старую таблицу конфигурации, не совпадающую с текущей сборкой игры.

## Минимальный тестовый сценарий

```text
map-list
restart
start
```

- `map-list` показывает технические имена доступных карт;
- `restart` загружает текущую карту заново и возвращает разминку;
- `start` начинает матч из разминки.

Полный список находится на странице [команд сервера](../commands/).
